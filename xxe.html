¿Qué es XXE?

XML External Entities es una vulnerabilidad que permite al atacante acceder a información mediante la interferencia de datos XML. Para poder realizarse el atacante debe tener acceso al archivo XML que se ejecuta y modificarlo añadiendo una entidad en la cual se usa la palabra clave SYSTEM y buscar un archivo con "file:" por la ruta absoluta o usar una url para acceder a otra máquina. El atacante puede leer archivos o realizar ataques SSRF.

Riesgos de la vulnerabilidad

  • Falsificación de peticiones al servidor
  • Ataque DoS
  • Robo de datos
